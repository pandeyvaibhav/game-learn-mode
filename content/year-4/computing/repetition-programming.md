# Repetition in Programming — Year 4 Computing

## Big Question
How do loops help us write shorter, more efficient programs?

## What We're Learning
In this lesson you will learn what **repetition** (loops) means in programming, how **count-controlled** loops work, and why loops make programs more **efficient**.

## Key Computing Words
| Word | What it means |
|---|---|
| loop | a set of instructions that repeats |
| repetition | doing something more than once |
| count-controlled loop | a loop that repeats a set number of times |
| infinite loop | a loop that never stops |
| efficiency | doing a task with less code or fewer steps |
| iteration | one pass through the loop |

## The Computing Explained

### Why Use Loops?
Without a loop, you must write the same instruction many times:
```
move forward
move forward
move forward
move forward
```

With a loop:
```
repeat 4 times:
    move forward
```

Both do the same thing — but the loop is **shorter** and **easier to change**.

### Count-Controlled Loops
A count-controlled loop runs a **fixed number of times**:
```
repeat 3 times:
    say "Hello!"
```
This says "Hello!" exactly **3 times**, then stops.

### Infinite Loops
An infinite loop repeats **forever** (until you stop the program):
```
forever:
    flash the light
```
Useful for things like a game's main loop or a flashing beacon.

### Loops Inside Loops (Nested)
You can put a loop inside another loop:
```
repeat 3 times:
    repeat 4 times:
        draw a side
    turn right
```

## Quick Quiz
1. What is a loop? → *A set of instructions that repeats*
2. What is a count-controlled loop? → *A loop that repeats a set number of times*
3. Why are loops useful? → *They make programs shorter and more efficient*
4. What is an infinite loop? → *A loop that never stops*

## Computing in the Real World
Loops are everywhere — traffic lights cycle through colours, playlists repeat songs, and washing machines loop through wash cycles. Every app and game uses loops.
