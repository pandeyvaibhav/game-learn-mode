# Variables in Programming 🎯

## What We're Learning
We're discovering variables — one of the most important ideas in programming. A variable is like a labelled box that stores information your program can use and change. You'll learn what data types are, how to name variables clearly, and how to update their values as a program runs.

## Key Words

| Word | Meaning |
|------|---------|
| variable | A named storage space in a program that holds a value which can change |
| data type | The kind of data stored in a variable, e.g. number, text (string), true/false (Boolean) |
| naming convention | A set of rules for giving variables clear, meaningful names |
| assignment | Setting or updating the value stored in a variable |

## Let's Explore! 🔍

### What Is a Variable?
Imagine a labelled box. The label is the variable's name and the contents are its value. In a game, you might have a variable called `score` that starts at 0 and goes up each time the player earns a point. The computer remembers the label and can look up or change the contents whenever your program tells it to.

### Data Types
Variables can hold different types of data. A **number** (integer or decimal) is used for scores, ages or prices. A **string** is text, like a player's name: `playerName = "Amira"`. A **Boolean** is either true or false, useful for things like `gameOver = false`. Choosing the right data type matters because the computer handles numbers, text and true/false values differently.

### Naming and Changing Values
Good variable names describe what the variable stores. Use names like `livesLeft` or `highScore` rather than `x` or `a`. In Scratch, you create a variable using the "Make a Variable" button. In text-based languages like Python, you simply write `score = 0`. You can change a variable's value during a program: `score = score + 10` adds 10 to whatever score already holds.

## Did You Know? 🌟
The word "variable" comes from the Latin *variabilis*, meaning "changeable." Early computer scientists chose this word because the value stored in a variable can change while a program runs!
