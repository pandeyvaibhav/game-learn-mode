# Variables in Programming — Year 5 Computing

## Big Question
What are variables and why are they important in programming?

## What We're Learning
In this lesson you will learn what a **variable** is, how to **name** variables, how to **set** and **change** their values, and why they are essential in programs.

## Key Computing Words
| Word | What it means |
|---|---|
| variable | a named container that stores a value in a program |
| value | the data stored inside a variable (number, text, true/false) |
| assign | to set or change the value of a variable |
| data type | the kind of data (e.g. number, string, boolean) |
| string | a data type for text (e.g. "hello") |
| boolean | a data type that is either true or false |

## The Computing Explained

### What Is a Variable?
Think of a variable as a **labelled box**. The label is the **name** and whatever is inside is the **value**.

```
score = 0        ← a variable called score, with value 0
name = "Ali"     ← a variable called name, with value "Ali"
```

### Why Use Variables?
- To **remember** information (e.g. a player's score).
- To **change** information as the program runs.
- To make programs **flexible** — the same code works with different data.

### Naming Rules
- Use clear, descriptive names: `playerScore` not `x`.
- No spaces — use camelCase (`playerScore`) or underscores (`player_score`).
- Start with a letter, not a number.

### Changing Values
```
score = 0
score = score + 10    ← score is now 10
score = score + 5     ← score is now 15
```

The old value is **replaced** each time you assign a new one.

### Data Types
| Type | Example |
|---|---|
| Number | 42, 3.14 |
| String | "Hello", "Year 5" |
| Boolean | true, false |

## Quick Quiz
1. What is a variable? → *A named container that stores a value*
2. What does "assign" mean? → *To set or change the value of a variable*
3. Give a good name for a variable storing a player's lives. → *playerLives (or similar descriptive name)*
4. What is a boolean? → *A data type that is either true or false*

## Computing in the Real World
Every app, game, and website uses variables. A shopping website stores your basket total in a variable. A game tracks your health, score, and level using variables.
