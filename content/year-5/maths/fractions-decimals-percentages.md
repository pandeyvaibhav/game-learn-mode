# Fractions, Decimals & Percentages — Year 5 Maths

## Big Question
How are fractions, decimals, and percentages connected?

## What We're Learning
In this lesson you will learn how to convert between **fractions**, **decimals**, and **percentages**, find **equivalent fractions**, and compare them.

## Key Maths Words
| Word | What it means |
|---|---|
| fraction | a part of a whole (e.g. ¾) |
| decimal | a number with a decimal point (e.g. 0.75) |
| percentage | a number out of 100 (e.g. 75%) |
| equivalent | equal in value (e.g. ½ = 0.5 = 50%) |
| numerator | the top number of a fraction |
| denominator | the bottom number of a fraction |

## The Maths Explained

### Key Equivalents
| Fraction | Decimal | Percentage |
|---|---|---|
| ½ | 0.5 | 50% |
| ¼ | 0.25 | 25% |
| ¾ | 0.75 | 75% |
| ⅕ | 0.2 | 20% |
| 1/10 | 0.1 | 10% |
| 1/100 | 0.01 | 1% |

### Converting
- **Fraction → Decimal**: divide numerator by denominator (e.g. 3 ÷ 4 = 0.75).
- **Decimal → Percentage**: multiply by 100 (e.g. 0.75 × 100 = 75%).
- **Percentage → Fraction**: write over 100 and simplify (e.g. 75% = 75/100 = ¾).

### Ordering
To compare fractions with different denominators, convert them to the **same denominator** or to decimals, then order.

Example: Order ½, ¾, ⅕ → 0.5, 0.75, 0.2 → **⅕, ½, ¾**

## Quick Quiz
1. What is ½ as a decimal? → *0.5*
2. What is 0.25 as a percentage? → *25%*
3. What is 75% as a fraction? → *¾*
4. Order these from smallest: ¾, ¼, ½ → *¼, ½, ¾*

## Maths in the Real World
Shop discounts use percentages (25% off). Recipes use fractions (½ cup). Money uses decimals (£3.75). Understanding all three helps with budgeting, cooking, and data.

## Learning Checklist
- [ ] I can convert between fractions, decimals, and percentages
- [ ] I know key equivalents (½ = 0.5 = 50%)
- [ ] I can order fractions by converting to decimals
