# Algebra — Year 6 Maths

## Big Question
How can we use letters and symbols to represent unknown numbers?

## What We're Learning
In this lesson you will learn about **expressions**, **equations**, **unknowns**, and simple **formulae**.

## Key Maths Words
| Word | What it means |
|---|---|
| algebra | using letters or symbols to represent unknown numbers |
| expression | a combination of numbers and letters, e.g. 2n + 3 |
| equation | a statement that two things are equal, e.g. 2n + 3 = 11 |
| unknown | the letter that represents a number we need to find |
| formula | a rule written using letters, e.g. area = l × w |
| variable | a letter that can take different values |

## The Maths Explained

### Expressions
An expression uses letters in place of numbers:
- **2n** means 2 × n
- **n + 5** means a number plus 5
- **3a − 2** means 3 times a, minus 2

### Equations
An equation has an **equals sign** and can be solved:
- n + 4 = 10 → n = 6
- 2n = 14 → n = 7
- 3n + 1 = 16 → 3n = 15 → n = 5

### Finding the Unknown
Use **inverse operations**:
1. Look at what has been done to the unknown.
2. Undo it in reverse order.

Example: 2n + 3 = 11
- Subtract 3: 2n = 8
- Divide by 2: n = 4

### Formulae
- Perimeter of a rectangle: P = 2(l + w)
- Area of a rectangle: A = l × w

If l = 5 and w = 3: A = 5 × 3 = 15

## Quick Quiz
1. What does 2n mean? → *2 × n*
2. Solve: n + 7 = 12 → *n = 5*
3. Solve: 3n = 18 → *n = 6*
4. If A = l × w, and l = 4, w = 6, what is A? → *24*

## Maths in the Real World
Algebra is used in science, engineering, computing, and finance. Every formula — from calculating speed to working out prices — uses algebra.
