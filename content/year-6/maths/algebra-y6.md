# Algebra 🎯

## What We're Learning
Algebra is like a puzzle where we use letters to stand for numbers we don't know yet. We'll learn how to write expressions, solve equations, and use formulae — skills that are the foundation of all higher mathematics.

## Key Words

| Word | Meaning |
|------|---------|
| expression | A combination of numbers, letters, and operations (e.g. 3n + 2) |
| equation | A statement that two things are equal, with an equals sign (e.g. 2x + 1 = 9) |
| unknown | A letter that represents a number we need to find |
| formula | A rule written using letters and numbers (e.g. area = length × width) |

## Let's Explore! 🔍

### Expressions and Variables
In algebra, we use letters to represent numbers. If we don't know how many sweets are in a bag, we can call it **n**. If we add 5 more sweets, we write **n + 5**. If we double the sweets, we write **2n** (which means 2 × n). Remember, in algebra we don't write the × sign between a number and a letter.

For example, if a cinema ticket costs £**p**, then 3 tickets cost **3p** and 3 tickets plus a £2 drink costs **3p + 2**.

### Solving Equations
An equation has an equals sign and an unknown value to find. Think of it like a balance — both sides must be equal.

To solve **2x + 3 = 11**:
1. What do we need to remove? The + 3. So subtract 3 from both sides: **2x = 8**
2. x is multiplied by 2, so divide both sides by 2: **x = 4**
3. Check: 2 × 4 + 3 = 11 ✓

### Formulae and Sequences
A formula lets us calculate something using a rule. For example, the perimeter of a rectangle is **P = 2l + 2w** where l = length and w = width.

We can also describe number sequences with algebra. The sequence 3, 6, 9, 12… follows the rule **3n** (multiply the position by 3). The 10th term would be 3 × 10 = 30.

## Try It Yourself! ✏️

**Question 1:** If n = 5, what is 4n − 3?
<details><summary>Answer</summary>4 × 5 − 3 = 20 − 3 = 17</details>

**Question 2:** Solve the equation: x + 7 = 15
<details><summary>Answer</summary>x = 15 − 7 = 8</details>

**Question 3:** Solve: 3y = 24
<details><summary>Answer</summary>y = 24 ÷ 3 = 8</details>

**Question 4:** A rectangle has length (n + 3) cm and width 4 cm. Write an expression for its perimeter.
<details><summary>Answer</summary>2(n + 3) + 2(4) = 2n + 6 + 8 = 2n + 14 cm</details>

**Question 5:** The rule for a sequence is 2n + 1. What are the first four terms?
<details><summary>Answer</summary>When n = 1: 3, n = 2: 5, n = 3: 7, n = 4: 9. The sequence is 3, 5, 7, 9.</details>

## Did You Know? 🌟
The word "algebra" comes from the Arabic word **al-jabr**, from the title of a book by the 9th-century mathematician al-Khwarizmi. He is often called the "father of algebra"!
